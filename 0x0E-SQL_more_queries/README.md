# SQL - More queries
Project done during **Software Engineering studies** at **AlX School**. It aims to learn about how to create a new user, manage privileges for a user, `PRIMARY KEY`, `FOREIGN KEY`, constraints, subqueries, `JOIN` and `UNION` with **MySQL**.

## Technologies
* `MySQL 8.0` (version 8.0.25)
* Tested on Ubuntu 20.04 LTS
